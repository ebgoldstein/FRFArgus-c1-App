# FRF ARGUS c1 Segemntation App

This is a deployed app to segement ARGUS imagery from FRF camera 1 (facing north). 

There are 4 classes: 
- Water + buildings+background
- Vegetation
- Sand
- Coarse sediment/Lag deposits
